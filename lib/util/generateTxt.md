# Usage


<a id="org86bdbc1"></a>

## `generateTxt`

`generateTxt` takes a richtext array and transforms it into a `Txt`. For instance, one can do:

    generateTxt(['Here is some normal text. ', i('But this is itallic.'), 'And we can even make text that ', g('GLOWS'), '.'])


<a id="orgeee150b"></a>

## `i`

Makes text itallic.


<a id="org18bf6d3"></a>

## `b`

Makes text bold.


<a id="org7a6da93"></a>

## `g`

Makes text glow.
